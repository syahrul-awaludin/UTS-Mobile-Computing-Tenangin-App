# Tenangin App 🧘‍♂️

Tenangin App is a modern, beautifully designed mobile application built with Flutter to help users manage their mental health and wellness. It provides daily affirmations, mood tracking, guided breathing exercises, and educational content to reduce stress and anxiety.

## ✨ Features

- **Mood Tracking**: Log your daily mood and view your emotional history.
- **Daily Affirmations**: Start your day with positive reinforcement.
- **Guided Exercises**: Take part in breathing exercises and mindfulness sessions.
- **Educational Content**: Watch curated YouTube videos and read articles about mental health.
- **Community**: Connect with others and share your journey.
- **Beautiful UI**: A highly polished, modern user interface with dynamic animations.

## 🚀 Getting Started

To run this project locally, ensure you have Flutter installed.

1. **Clone the repository**
   ```bash
   git clone https://github.com/syahrul-awaludin/UTS-Mobile-Computing-Tenangin-App.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd UTS-Mobile-Computing-Tenangin-App
   ```

3. **Install dependencies**
   ```bash
   flutter pub get
   ```

4. **Run the app**
   ```bash
   flutter run
   ```

## 🛠 Tech Stack

- **Framework**: Flutter
- **Language**: Dart
- **Video Player**: `youtube_player_iframe`
